# §2 Definition of Ready｜開工前的 PRD 健檢

> DoD 守「做完沒」，DoR 守「這份規格能不能開始做」。兩者是對偶：DoR 在入口、DoD 在出口。

## 本章涵蓋（v0.5.0 / Phase 2 已交付）

- [§2.1 DoR 可實作性健檢清單](01_dor_checklist.md) — 逐端點 / 逐表 / 認證歸屬 / NFR / 邊界 / 驗收標準的健檢清單，三級制快檢 vs 全檢
- [§2.2 PRD 健檢報告與不合格處置](02_prd_health_report.md) — 健檢報告格式 + 不完美 PRD 的 go/no-go 決策樹（健康 / 有小洞標假設開工 / 有 blocker 退回）

## 何時載入

- 拿到一份新 PRD / 規格，動工前
- 接手一個 PRD 區塊，不確定它寫得夠不夠清楚
